Cards and Roses

Cards and Roses is an immersive card-based adventure game developed in Unity Engine. This project showcases my skills in game mechanics design, user interface development, and creating engaging gameplay systems.

The game features strategic card gameplay combined with narrative elements, where players navigate through various challenges using a carefully crafted deck system. I focused on creating intuitive game mechanics and polished user experience.

This project allowed me to explore advanced Unity features including UI system design, game state management, and interactive card mechanics. The rose theme was woven throughout the visual design to create a cohesive aesthetic experience.

Key achievements include implementing complex card interactions, designing responsive UI elements, and creating smooth gameplay flow that keeps players engaged throughout their journey.

Development Highlights

Key Features

The development process focused on creating engaging card-based mechanics that feel intuitive and rewarding. Using Unity's powerful systems, I implemented complex card interactions and smooth gameplay flow.

The rose-themed visual design was carefully integrated throughout the game, from card artwork to UI elements, creating a cohesive and atmospheric experience that enhances the gameplay narrative.

Special attention was paid to user experience design, ensuring that card mechanics are easy to learn but offer depth for strategic gameplay, appealing to both casual and experienced players.
