Cards and Roses

Cards and Roses is a card battle game developed in Unity Engine. This project showcases my skills in game mechanics design, user interface development, and designing engaging gameplay systems.

The game features strategic card gameplay combined with narrative elements, where players navigate through various challenges using a carefully randomized deck system. I focused on creating intuitive game mechanics and polished user experience.

This project allowed me to really learn unity since before this i had only done 1 small project which did not require much technical prowess so this was a big step. The game had a visual theme centered around greek mythology specifically the story of Eros and Psyche.

Key achievements include creating the entire deck system, all of the logic surrounding how when and where you can play cards, incorporating the artist's art into the game, designing the gameplay with the game artists, giving every card their own data to be used for the other features and lastly i created most of the enemy ai which was sadly not finished in time due to how complicated the mechanics were and my lack of knowledge in ai creation at the time.

Development Highlights

Key Features

The development process focused on creating the mechanics required for a card game. Using Unity's systems such as interactive objects, I implemented complex card mechanics and smooth gameplay flow.

The ancient Greek themed visual design was carefully integrated throughout the game, from card artwork to UI elements, creating a cohesive and atmospheric experience that enhances the gameplay narrative.

Special attention was paid to user experience design, ensuring that card mechanics are easy to learn but offer depth for strategic gameplay, appealing to both casual and experienced players.

The project ended up not being finished due to the time constraints of the assignment, but it served as a valuable learning experience in game development, project management, and teamwork.

public class PlayerDeck : MonoBehaviour
{
    public GameObject CardPrefab;
    public GameObject Hand;
    [SerializeField]private List<CardSettings> deck;
    List<CardSettings> runDeck = new List<CardSettings>();
    public int beginningHand = 0;
    private List<GameObject> cardsInHand = new List<GameObject>();
    public GameObject CardToHand;
    public GameObject cardInDeck1;
    public GameObject cardInDeck2;
    public GameObject cardInDeck3;
    public GameObject cardInDeck4;

    void Start()
    {
        Shuffle(deck);
        StartCoroutine(StartGame());
    }
    
    private void Update()
    {
        DeckVisuals();
    }
    
    public void DrawCard(int cardIndex)
    {   
        var NewCard = Instantiate(CardToHand);
        NewCard.transform.position = new Vector3(-100, 0, 0);
        cardsInHand.Add(NewCard);
        DisplayCard dcs = NewCard.GetComponent<DisplayCard>();
        dcs.Init(runDeck[cardIndex]);
        runDeck.RemoveAt(cardIndex);
    }

    public void Shuffle(List<CardSettings> deck)
    {
        runDeck = deck.OrderBy(x => Random.value).ToList();
    }

    IEnumerator StartGame()
    {
        for (int i = 0; i < beginningHand; i++)
        {
            DrawCard(0);
            yield return new WaitForSeconds(1);
        }
    }
    
    public void DeckVisuals()
    {
        if (runDeck.Count < 20) cardInDeck1.SetActive(false);
        if (runDeck.Count < 13) cardInDeck2.SetActive(false);
        if (runDeck.Count < 7) cardInDeck3.SetActive(false);
        if (runDeck.Count < 1) cardInDeck4.SetActive(false);
    }
}

    private int Randomizer(int Min, int Max)
    {
        int number = UnityEngine.Random.Range(Min, Max);
        return number;
    }

    IEnumerator AIBehaviour()
    {
        for (int i = 0; i < numberofactions; i++)
        {
            int number = Randomizer(1, 4);

            if (EnemyCardsinRanged.Count == 0)

                }
                else

                if (movedmeleecard != null)

                    MoveCardToMelee();
                }
                else
                {
                    deck.DrawCard(0);
                }
            } 
        }
        turn.EndyourOponentTurn();
    }

    void handleOnEndTurn() 
    {
        if (turn.isYourTurn == false)
        {
            StartCoroutine(AIBehaviour());
        }
        else if (turn.isYourTurn == true)
        {
            StopCoroutine(AIBehaviour());
        }
    }
